## 1. 产品概述

### 1.1 产品定位

一款轻量级的心情记录工具，帮助用户记录生活事件与情绪反应，通过心理学知识分析情绪；同时提供"小确幸"清单功能，帮助用户积累愉悦身心的小事，通过完成小事提升心情，追踪记录完成后的情绪变化。

**产品slogan**：记录心情，遇见更好的自己

### 1.2 核心价值

| 核心价值 | 说明 |
|----------|------|
| **情绪觉察** | 帮助用户识别和命名自己的情绪，提升情绪智力 |
| **情绪分析** | 基于事件类型分析情绪模式，发现情绪规律 |
| **心理调适** | 提供专业的心理学支持和建议，科学应对负面情绪 |
| **个性记忆** | 记住用户偏好和行为模式，提供越来越懂你的服务 |
| **深度洞察** | 透过情绪表象，分析深层心理需求和模式 |

### 1.3 目标用户

| 用户画像 | 特征 |
|----------|------|
| 职场新人 | 18-30岁，初入职场，面临工作压力和角色转换 |
| 焦虑人群 | 25-40岁，工作/生活压力大，需要情绪管理工具 |
| 心理成长者 | 20-35岁，对心理学感兴趣，希望自我探索和成长 |
| 冥想爱好者 | 20-45岁，练习正念冥想，需要记录心情变化 |

### 1.4 产品特点

**三大创新点**：

1. **心理学引擎** - 本地规则引擎 + AI增强，提供专业的心理分析
2. **小确幸系统** - 积累愉悦小事，完成后追踪心情变化，形成正向循环
3. **双重记录模式** - "选择心情"快速记录，或"心情随笔"智能分析复杂情绪

## 3. 功能需求详述

### 3.1 模块一：心情记录（核心模块）

#### 3.1.1 两种记录模式

**模式一：选择心情**

| 功能点 | 描述 |
|--------|------|
| 心情选择 | 7种心情类型网格展示，emoji + 文字 |
| 强度调节 | 1-10滑块调节 |
| 事件描述 | 可选填写，最少10字符 |
| 标签选择 | 多选标签（工作/家庭/健康等） |
| 心理学建议 | 基于选择的心情展示专业建议 |

**心情类型定义**：

| 心情 | Emoji | 关键词 |
|------|-------|--------|
| 开心 | 😊 | 高兴、快乐、幸福、愉悦 |
| 平静 | 😌 | 放松、宁静、安心、舒适 |
| 焦虑 | 😰 | 担心、紧张、压力、不安 |
| 伤心 | 😢 | 难过、悲伤、失落、沮丧 |
| 生气 | 😠 | 愤怒、恼火、烦躁、不满 |
| 害怕 | 😨 | 恐惧、担心、害怕、惶恐 |
| 惊讶 | 😲 | 意外、震惊、惊喜、诧异 |

**模式二：心情随笔**

| 功能点 | 描述 |
|--------|------|
| 事件输入 | 多行文本输入，最少10字符触发分析 |
| 智能分析 | 基于关键词的情绪分析 |
| 混合心情 | 支持多种心情混杂，显示占比 |
| 心理评估师 | 生成专业、文艺的心理洞察 |
| 强度智能 | 自动分析并允许手动调整 |

**记录模式对比**：

| 维度 | 选择心情 | 心情随笔 |
|------|----------|----------|
| 适用场景 | 情绪明确 | 情绪复杂 |
| 操作复杂度 | 简单 | 中等 |
| 分析方式 | 用户自选 | AI智能分析 |
| 事件要求 | 可选 | 必须 |
| 混合心情 | 不支持 | 支持 |

#### 3.2.2 小确幸分类

| 分类 | 示例小事 | 频率建议 |
|------|----------|----------|
| 放松 | 泡一杯茶、听轻音乐 | daily |
| 运动 | 散步、瑜伽、跑步 | daily/weekly |

| 自然 | 户外散步、养花 | weekly |

#### 3.2.3 推荐算法

**推荐规则**：

1. **心情匹配** - 根据用户当前心情推荐适合的小事
2. **频率去重** - daily类小事每天不重复推荐
3. **效果排序** - 根据历史完成后的心情改善效果排序
4. **偏好学习** - 记住用户喜好，优先推荐喜欢的分类

**心情-小事匹配矩阵**：

| 当前心情 | 推荐分类 | 理由 |
|----------|----------|------|
| 开心 | 所有分类 | 强化正向情绪 |
| 平静 | 放松、感官、自然 | 维持平静状态 |
| 焦虑 | 放松、运动、感官 | 缓解焦虑 |
| 伤心 | 社交、创作、自然 | 转移注意，提升情绪 |
| 生气 | 运动、整理、感官 | 释放能量 |
| 害怕 | 放松、社交、成长 | 获取安全感 |
| 惊讶 | 放松、自然、感官 | 稳定情绪 |

### 3.5 模块五：深层心理分析（洞察报告）

#### 3.5.1 马斯洛需求层次分析

**分析维度**：

| 需求层次 | 对应情绪信号 | 相关标签 |
|----------|--------------|----------|
| 生理需求 | 疲惫、烦躁（未满足）；精力充沛（已满足） | health |
| 安全需求 | 焦虑、恐惧（未满足）；平静、放松（已满足） | work, finance |
| 社交需求 | 孤独、悲伤（未满足）；温暖、归属（已满足） | family, relationship |
| 尊重需求 | 愤怒、羞耻（未满足）；自信、自豪（已满足） | work |
| 自我实现 | 空虚、迷茫（未满足）；充实、有意义（已满足） | growth, study |

**可视化**：雷达图展示各层次满足度

#### 3.5.2 心理防御机制识别

**识别类型**：

| 机制 | 成熟度 | 识别信号 |
|------|--------|----------|
| 升华 | 成熟 | 负面情绪后有正向行动（运动、创作） |
| 幽默 | 成熟 | 能自嘲、在压力下开玩笑 |
| 压抑 | 中间 | 情绪强度较低、回避特定话题 |
| 合理化 | 中间 | 过度解释、为情绪找逻辑理由 |
| 否认 | 不成熟 | 言行与记录情绪不符 |

#### 3.5.3 情绪触发因素分析

**触发类型**：

| 类型 | 分析内容 | 输出 |
|------|----------|------|
| 事件触发 | 特定事件引发的情绪 | "工作汇报 → 焦虑" |
| 时序触发 | 情绪之间的转换规律 | "焦虑 → 愤怒" |
| 社交触发 | 人际关系引发的情绪 | "与家人 → 平静" |
| 环境触发 | 时间/环境引发的情绪 | "周末 → 开心" |

#### 3.5.4 心理成长追踪

**追踪指标**：

| 指标 | 计算 | 趋势 |
|------|------|------|
| 情绪稳定性 | 强度方差 | 下降 = 进步 |
| 恢复时间 | 负面情绪持续天数 | 缩短 = 进步 |
| 正向情绪占比 | 积极情绪 / 总情绪 | 上升 = 进步 |
| 小确幸效果 | 完成后心情改善率 | 上升 = 进步 |

## 5. 数据模型

### 5.1 心情记录

```typescript
interface MoodRecord {
  id: string;
  event: string;
  mood: MoodType;
  intensity: number;       // 1-10
  tags: Tag[];
  createdAt: string;       // ISO timestamp
  smallHappyId?: string;   // 关联的小确幸（可选）
  mixedMoods?: MixedMood[]; // 混合心情（可选）
  analysis?: {
    insight?: string;       // 心理评估师洞察
    needLevel?: NeedLevel;  // 需求层次
  };
}
```
