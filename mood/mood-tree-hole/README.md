# 🌳 心情树洞 (Mood Tree Hole)

> 一款融合心理学知识与积极心理学的智能心情记录应用

## ✨ 功能特性

- **双模式心情记录** - 选择心情 / 心情随笔
- **小确幸清单系统** - 智能推荐 + 完成追踪
- **心理学分析引擎** - 马斯洛需求 / 防御机制 / 触发因素
- **数据可视化** - 心情曲线 / 热力图 / 雷达图
- **完全本地化** - localStorage 存储，隐私无忧

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📦 技术栈

- React 17.0.2
- TypeScript 4.9
- Vite 4.2.0
- Tailwind CSS 3.2.0
- Recharts 2.5.0

## 📖 项目结构

```
mood-tree-hole/
├── src/
│   ├── components/      # 组件库
│   ├── pages/           # 页面
│   ├── context/         # 状态管理
│   ├── data/            # 数据层
│   ├── utils/           # 工具函数
│   └── types/           # 类型定义
├── public/              # 静态资源
└── package.json         # 依赖配置
```
